API Documentation - NounCity

Authentication

Login

Endpoint: POST https://nouncity.com/api/login
Description: Authenticate user and obtain access token.

Request Body:

{
  "phone": "085702053522",
  "password": "password"
}

Response:
- 200 OK - Login successful, returns user token and details.
- 401 Unauthorized - Invalid credentials.

Trips

List Trips

Endpoint: GET https://nouncity.com/api/admin/trips
Description: Retrieve a list of all trips.

Response:
- 200 OK - Returns array of trip objects.

Get Single Trip

Endpoint: GET https://nouncity.com/api/admin/trips/{trip_id}
Description: Retrieve details of a single trip by ID.

Path Parameters:
- trip_id (integer) - ID of the trip.

Response:
- 200 OK - Returns trip object.
- 404 Not Found - Trip not found.

Create Trip

Endpoint: POST https://nouncity.com/api/admin/trips
Description: Create a new trip.

Request Body:

{
  "date": "2025-05-8",
  "trip_type_id": 1,
  "boat_id": 5,
  "remarks": "testing boatman create trip",
  "status": "scheduled",
}

Response:
- 201 Created - Trip successfully created.
- 400 Bad Request - Validation errors.

Update Trip

Endpoint: PUT https://nouncity.com/api/admin/trips/{trip_id}
Description: Update details of an existing trip.

Path Parameters:
- trip_id (integer) - ID of the trip to update.

Request Body:

{
  "date": "2025-05-07",
  "trip_type_id": 1,
  "boat_id": 5,
  "remarks": "test request on postman update",
  "status": "scheduled"
}

Response:
- 200 OK - Trip successfully updated.
- 404 Not Found - Trip not found.
- 400 Bad Request - Validation errors.

Complete Trip

Endpoint: POST https://nouncity.com/api/trips/{trip_id}/complete
Description: Mark a trip as completed.

Path Parameters:
- trip_id (integer) - ID of the trip.

Response:
- 200 OK - Trip marked as completed.

Cancel Trip

Endpoint: POST https://nouncity.com/api/trips/{trip_id}/cancel
Description: Cancel a trip.

Path Parameters:
- trip_id (integer) - ID of the trip.

Response:
- 200 OK - Trip canceled.
- 404 Not Found - Trip not found.

Search Trips

Endpoint: GET https://nouncity.com/api/trips/search
Description: Retrieve a list of trips filtered by optional parameters.

Query Parameters:

  • date (date) – Date of the trip (YYYY-MM-DD)
  • trip_type_id (integer) – ID of the trip type
  • boat_id (integer) – ID of the boat
  • boatman_id (integer) – ID of the boatman
  • status (string) – Trip status

Options for status:

  • scheduled – Trip is scheduled
  • completed – Trip has been completed
  • cancelled – Trip was cancelled

Example Request:

GET https://nouncity.com/api/trips/search?date=2025-05-20&boat_id=2&status=scheduled

Response:
- 200 OK - Returns a list of filtered trips.
- 400 Bad Request - Invalid parameters.

Search Tickets

Endpoint: GET https://nouncity.com/api/tickets/search
Description: Retrieve a list of tickets filtered by optional parameters such as hotel_id, payment_method, and payment_status.

Query Parameters:

  • hotel_id (integer) – ID of the hotel
  • payment_method (string) – Method of payment
  • payment_status (string) – Status of payment

Options for payment_method:

  • cash – Paid using physical cash
  • credit_card – Paid using a credit card
  • bank_transfer – Paid via bank transfer

Options for payment_status:

  • pending – Payment not completed yet
  • paid – Payment has been successfully completed

Example Request:

GET https://nouncity.com/api/tickets/search?hotel_id=4&payment_method=bank_transfer&payment_status=paid

Response:
- 200 OK - Returns a list of filtered tickets.
- 400 Bad Request - Invalid parameters.

Send Invoice

Endpoint:
POST https://nouncity.com/api/invoice/send/{invoice_id}

Description:
Sends an invoice email with a PDF attachment to the associated hotel. The trip must be marked as completed, and there must be no passengers who paid by cash.

Path Parameters:

  • invoice_id (integer) – ID of the invoice to be sent.

Responses:

  • 200 OK – Invoice email sent successfully.
  • 400 Bad Request – Trip is not completed, has cash-paid passengers, or hotel email is missing.
  • 404 Not Found – Invoice or hotel not found.
